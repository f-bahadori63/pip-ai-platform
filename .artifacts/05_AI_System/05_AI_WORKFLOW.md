# Purpose

---

# AI Request Flow

## Step 1 — User Request

User sends a request through PIP interface.

Examples:

- Analyze project delay

- Review risks

- Generate report

---

## Step 2 — Request Understanding

AI system identifies:

- User intention

- Required information

- Relevant agent

---

## Step 3 — Agent Selection

The AI Orchestrator selects the appropriate agent.

Examples:

- Planning Agent

- Risk Agent

- Document Agent

- Cost Agent

---

## Step 4 — Knowledge Retrieval

System retrieves required information from:

- Project database

- Documents

- Knowledge base

- Historical data

---

## Step 5 — AI Processing

AI performs:

- Analysis

- Reasoning

- Recommendation generation

---

## Step 6 — Response Generation

Output is prepared as:

- Answer

- Report

- Recommendation

- Warning

---

## Step 7 — User Feedback

User feedback is collected for:

- Improvement

- Validation

- Future learning
