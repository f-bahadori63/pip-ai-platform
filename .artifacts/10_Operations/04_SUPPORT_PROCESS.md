# Support Levels

## Level 1 Support

Responsible for:

- User questions

- Basic issues

- Access problems

---

## Level 2 Support

Responsible for:

- Application issues

- Configuration problems

- Functional errors

---

## Level 3 Support

Responsible for:

- Technical failures

- Code issues

- Architecture problems

# Incident Management

Process:

1. Register incident

2. Analyze issue

3. Resolve problem

4. Document solution
